HTTP Запросы:
----------------------------------------------------------------------------------------------------------------
GET    - получает данные с сервера    GET/api/users   Не меняет данные   Безопасный     Идемпотентный
----------------------------------------------------------------------------------------------------------------
POST   - создает новый объект         POST/api/user   Меняет данные      Не безопасен   Не идемпотентный
----------------------------------------------------------------------------------------------------------------
PUT    - обновление объект            PUT/api/user    Меняет данные      Не безопасен   Идемпотентный
---------------------------------------------------------------------------------------------------------------
PATCH  - частичное обновление         PATCH/api/user  Меняет данные                     Идемпотентный: да и нет
         объекта не нужно отправлять
         весь объект
----------------------------------------------------------------------------------------------------------------
DELETE - удаление объекта (по                                                           Идемпотентный:
         идентификатору)                                                                необратимое действие
----------------------------------------------------------------------------------------------------------------

Идемпотентность - количество раз, что можно безопасно выполнить запрос (напр, не создаются новые данные, а изменяются)

200 - OK                    Все успешно
201 - Created               Объект успешно создан (POST)
400 - Bad request           Ошибка в данных
404 - Not found             Объект не найден
500 Internal Server Error   Ошибка сервера
